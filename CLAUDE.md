# Brain — Claude Context

This repository is a shared knowledge base of markdown files that defines how the team works together.

## What this repo contains

- `collaboration/` — norms around communication, branching, and code review
- `standards/` — language and testing conventions
- `processes/` — step-by-step workflows for releases, onboarding, incidents

## Working in this repo

- All content is markdown. Keep files focused — one topic per file.
- CI checks for broken links and markdown formatting issues.
- Run `python tools/check_links.py` locally before pushing to catch broken internal links.
- See `CONTRIBUTING.md` for the full contribution workflow.
