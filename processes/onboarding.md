# Onboarding

## Day 1

- [ ] Get access to the repo and clone it
- [ ] Read `README.md`, `CONTRIBUTING.md`, and `CLAUDE.md`
- [ ] Set up your local Python environment (`python -m venv .venv && pip install -e ".[dev]"`)
- [ ] Meet your onboarding buddy

## Week 1

- [ ] Read all files in `collaboration/` and `standards/`
- [ ] Open a small PR (fix a typo, add a clarification) to get familiar with the workflow
- [ ] Attend a team sync

## Done when

You can open, review, and merge a PR without asking for help.
