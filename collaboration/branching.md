# Branching

Branch off `main`. Use a short, descriptive name with a prefix:

| Prefix | Use for |
|---|---|
| `feat/` | New content or features |
| `fix/` | Corrections to existing content |
| `update/` | Revisions to norms or processes |
| `chore/` | Tooling, CI, repo maintenance |

Delete branches after merging.
