# Communication

## Defaults

- Prefer async over sync. Write things down.
- If a conversation takes more than 5 back-and-forths, move it to a call.
- Decisions made verbally get documented in a PR or file within 24 hours.

## Where things live

| Topic | Where |
|---|---|
| Technical decisions | PR description or a file in this repo |
| Quick questions | Chat (Slack / Teams / etc.) |
| Incidents | `processes/incidents.md` |
