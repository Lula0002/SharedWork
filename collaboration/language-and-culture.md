# Language and Culture

We work across English and Danish. These norms reduce friction from language barriers and cultural differences.

## Language defaults

- All written artifacts — PRs, docs, issues, comments — are in **English**.
- Danish is fine for quick chat, but any decision that gets documented must be in English so nothing is siloed.
- In mixed meetings: speak slowly, avoid slang, and repeat key decisions in writing after the call.

## Writing for a mixed audience

- Be literal. Idioms, sarcasm, and deadpan humor don't cross well in writing.
- Async communication favors non-native speakers — they have time to compose and read carefully. Lean into it.
- If a written message feels ambiguous, assume good intent and ask directly rather than inferring tone.

## Cultural differences to be aware of

### Directness

Danish culture is direct — disagreement is expressed plainly regardless of seniority. English-speaking cultures (especially US) often soften pushback with hedging phrases. A Dane skipping the softening is not being rude.

The flip side: if feedback feels harsh, check whether it's a style difference before treating it as a conflict.

### Hierarchy

Danish workplaces are flat. Anyone is expected to challenge an idea, including a senior person's. If you're used to a more hierarchical culture, this is a feature — use it.

### Consensus vs. speed

Danish culture tends toward talking until everyone agrees. Some English-speaking cultures are comfortable with one person deciding. Agree explicitly on which mode you're in for a given decision. The RACI framework in [communication.md](communication.md) helps here.

### Self-promotion

Danish culture discourages visible self-promotion (Janteloven). This can look like underselling work or not advocating loudly for ideas. If someone's contribution isn't visible, ask them directly — don't wait for them to surface it.

### "It's fine"

Danish understatement means "it's fine" can signal unexpressed concerns. For important decisions, use explicit written sign-offs rather than relying on silence as agreement.
