# Brain

A shared knowledge repository defining how we work together — our processes, standards, and collaboration norms. Versioned in git so the whole team can pull, use, and improve it over time.

## Structure

| Folder | Purpose |
|---|---|
| `collaboration/` | How we communicate, review, and coordinate |
| `standards/` | What good code and documentation looks like |
| `processes/` | Step-by-step workflows for recurring tasks |
